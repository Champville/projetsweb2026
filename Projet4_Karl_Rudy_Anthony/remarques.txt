Fonctions Globales
saveTimerState(startTime, duration) : Enregistre l'heure de début et la durée du quiz dans le stockage local du navigateur.

loadTimerState() : Récupère les données du minuteur enregistrées pour reprendre là où l'utilisateur s'était arrêté.

clearTimerState() : Supprime les données du minuteur du stockage local.

(Fonction anonyme au début) : Initialise immédiatement le thème (clair ou sombre) dès le chargement de la page.

syncThemeBtn() : Met à jour le texte et l'icône du bouton de changement de thème selon le mode actif.

initBinaryRain() : Initialise et gère l'animation visuelle de "pluie binaire" sur le canvas.

resize() : Ajuste les dimensions du canvas d'animation lorsque la fenêtre est redimensionnée.

draw() : Dessine les caractères "0" et "1" qui tombent pour créer l'effet de pluie.

initConverter() : Configure les écouteurs d'événements pour l'outil de conversion Décimal/Binaire.

initQuiz() : Fonction principale qui gère toute la logique du quiz (questions, score, minuteur).

initPersistentTimer() : Lance ou restaure le compte à rebours en fonction du stockage local.

updateTimerDisplay() : Calcule le temps restant et met à jour l'affichage visuel du chronomètre.

endQuizDueToTime() : Termine automatiquement le quiz lorsque le temps est écoulé.

renderQ() : Affiche la question actuelle et génère les boutons d'options correspondants.

finishQuiz() : Calcule le score final, arrête le temps et affiche les résultats.

Variables Globales et Constantes
QUESTIONS : Tableau d'objets contenant les énoncés, les options et les bonnes réponses du quiz.

saved : Stocke la préférence de thème (light/dark) récupérée dans le localStorage.

btn : Référence au bouton HTML qui permet de basculer entre le mode clair et sombre.

Variables Locales Clés (dans les fonctions)
canvas / ctx : Utilisés pour manipuler la zone de dessin et le contexte 2D de l'animation.

drops : Tableau qui stocke la position verticale de chaque colonne de la pluie binaire.

decInput / binInput : Champs de saisie pour les nombres décimaux et binaires du convertisseur.

curr : Index de la question actuellement affichée dans le quiz.

answers : Tableau stockant les réponses sélectionnées par l'utilisateur.

timerInterval : Stocke l'identifiant de l'intervalle pour pouvoir arrêter le chronomètre plus tard.

finished : Booléen indiquant si le quiz est terminé ou toujours en cours.

score : Nombre total de réponses correctes après calcul final.
