Architecture et systèmes d'exploitation

Introduction

  • Un ordinateur est une machine qui traite des informations
  • Il est composé de matériel (hardware) et de logiciels (software)
  • Le système d’exploitation permet de faire fonctionner la machine

Frise chronologique

1940

Alan Turing aide à décrypter Enigma pendant la Seconde Guerre mondiale

1945

ENIAC : premier ordinateur électronique utilisé pour les calculs

1950

Architecture de Von Neumann utilisée dans tous les ordinateurs

1970

Apparition des microprocesseurs

Aujourd’hui

Ordinateurs rapides avec systèmes modernes

Éléments importants

  • Processeur (CPU)
  • Mémoire (RAM, disque dur)
  • Périphériques (clavier, écran)
  • Système d’exploitation
