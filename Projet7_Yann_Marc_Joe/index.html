L'Histoire de Python ?

Un peu d'histoire...

Tout débutte avec un certain mathématicien nommé Guido van Rossum.

Ce dernier:

  1. Était un passionné de langages de programmation.
  2. Travaillait au laboratoire Centrum Wiskunde & Informatica (CWI) aux Pays-Bas.
  3. Aimait les Monty Python, une troupe de comédie durant les années 1970.

Pour cela, il a décidé de créer un langage de programmation à la fois facile à utiliser et lisible.

Ainsi durant les vacances de Noël en 1989, il nomme son langage de programmation ABC.

Cependant, en raison de son amour pour Monty Python, il crée un nouveau langage comme successeur à ABC qu'il renomme vite Python.

Guido van Rossum


Comment a-t-il commencé ?

Vers la fin des années 1870, Guido van Rossum a créé Python dans le laboratoire Centrum Wiskunde & Informatica (CWI) aux Pays-Bas.

Il a commencé à travailler sur Python en 1989, et la première version publique de Python, la version 0.9.0, a été publiée en 1991. (De nos jours, la version la plus récente est 3.14.4).

Python contient déjà des concepts modernes comme: classes, exceptions, fonctions, types de données avancés (listes, dictionnaires).

Le langage devient populaire avec la sortie de Python 2.x en 2000.

En 2008, Python 3.x sort pour corriger des problèmes structurels de Python 2.x


Évolution de Python en bref:

Version de Python Changements et nouveautés
1.0 Outils de programmation fonctionnels (lambda, map, filter)
2.0 Introduction des classes, des exceptions et des listes en compréhension
3.0 Incompatibilité avec les versions précédentes mais amélioration de la syntaxe et des fonctionnalités
3.x Les versions les plus récentes de Python sont encore dans Python 3.x

Et Aujourd'hui ?

Python est aujourd'hui souvent le premier langage appris par les débutants (nous sommes un exemple de cela).

Mais il est aussi utilisé par des professionnels et des organisations très majeures comme Google, OpenAI ou la NASA.
