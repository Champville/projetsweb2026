C'est quoi, Python ?

Langage de Programmation

Python est un langage de programmation qui permet de communiquer avec un ordinateur. Imagine que tu donnes des instructions à ton ordinateur dans une langue qu'il peut comprendre.

Permet la Communication avec l'ordinateur

  1. Le programmeur conçoit et écrit un code pour son programme souhaité.
  2. Un fichier texte écrit dans une langue proche de l'Anglais.
  3. Instructions binaires éxécutés par l'ordinateur.

Pourquoi apprendre Python ?

Python est idéal pour les débuttants car il est simple à lire et à écrire. De plus, c'est l'un des langages les plus utilisés dans le monde professionnel, notamment pour l'intelligence artificielle, l'analyse de données, et les sites web.

Variables dans Python

C'est quoi une variable?

Une variable, c'est comme une boîte avec une étiquette. Tu peux écrire une valeur dedans et l'utiliser ou même changer son contenu de nouveau.

Créer et utiliser une variable

Pour créer une variable, tu lui donnes un nom et tu y stockes une valeur:

age = 15
nom = "Marie"
taille = 1.65

Modifier une variable

Tu peut changer la valeur d'une variable à tout moment:

score = 0
score = score + 10
score += 5

À la fin, score vaut 15.

Affichage

La fonction print()

Pour afficher quelque chose à l'écran, utilise print() :

message = "bonjour"
print(message)

Résultat: bonjour

Tu peux afficher plusieurs choses en les séparant par des virgules :

prenom = "alex"
age = 16
print("Je m'appelle", prenom, "et j'ai", age, "ans")

Résultat: Je m'appelle alex et j'ai 16 ans

Les f-string

Une façon moderne et pratique d'insérer des variables dans du texte.

Ajoute un f avant les guillemets :

ville = "Paris"
temperature = 22
print(f"Il fait {temperature}°C à {ville} aujourd'hui")

Résultat: Il fait 22°C à Paris aujourd'hui

Un autre exemple avec arrondissement :

prix = 42.46587
print(f"Le total est {prix:.1f}€")

Résultat: Le total est 42.5€

Caractères spéciaux

Dans une chaine de caractères :

  • Le caractère \n crée une nouvelle ligne dans ton texte.
  • Le caractère \t représente un espace TAB.
  • Le caractère \\ est utilisé pour afficher un backslash \.

Fonction

Une fonction est comme une machine

Tu lui donnes des ingrédients (arguments), elle fait un traitement, et elle te rend un résultat.

Créer une fonction

Pour définir une fonction, on utilise le mot-clé def:

def calculer_carre(nombre):
    resultat = nombre * nombre
    return resultat

Appeler une fonction

Une fois définie, tu peux appeler ta fonction autant de fois que tu veux :

mon_nombre = 5
carre = calculer_carre(mon_nombre)
print(carre)

Résultat: 25

Fonctions avec plusieurs paramètres

Une fonction peut accepter aucun ou plusieurs arguments :

def calculer_aire_rectangle(longueur, largeur):
    return longueur * largeur

print(calculer_aire_rectangle(10,2))

Résultat: 20

Valeurs par défaut

Paramètres par défault dans une fonction

Tu peux donner des valeurs par défaut à certains paramètres. Si tu n'indiques pas de valeur lors de l'appel, Python utilisera la valeur par défaut :

def saluer(nom, formule="Bonjour"):
    return f"{formule} {nom}"

print(saluer("Julie")) # Utilise "Bonjour"
print(saluer("Paul", "Salut")) # Utilise "Salut"

Résultat: Bonjour Julie

Résultat: Salut Paul

Exemple pratique : calcul de prix

Imaginons une fonction qui calcule le prix TTC (avec TVA). Au Liban, la TVA est de 11% par défaut :

def prix_ttc(prix_ht, tva=11):
    return prix_ht * (1 + tva/100)

# Utilise la TVA par défaut (11%)
print(prix_ttc(100))
# Affiche: 111.0

# Spécifie une TVA différente (20%)
print(prix_ttc(200, 20))
# Affiche: 240.0

# On peut aussi nommer les paramètres
print(prix_ttc(tva=12, prix_ht=300)) # Affiche: 336.0
print(prix_ttc(prix_ht=100)) # Affiche: 111.0

ATTENTION

Les paramètres avec valeurs par défaut doivent toujours venir après les paramètres obligatoires dans la définition de la fonction.

return vs print : return renvoie une valeur, print l'affiche. Ce n'est pas pareil !

Encapsulation : Une fonction doit être indépendante. N'utilise JAMAIS des variables définies en dehors de la fonction !

input() : N'utilise pas input() dans une fonction. Passe plutôt la valeur en argument. Pourquoi ? Parce que ce n'est pas professionnel !

Expressions booléennes

Une expression booléenne est une question qui a une réponse : True (vrai) ou False (faux).

Une expression booléenne peut être obtenu en Python à partir des opérateurs de comparaison et les opérateurs logiques.

Opérateurs de comparaison

Opérateur Fonction
== Strictement Égal
!= Différent
> Strictement Supérieur
< Strictement Inférieur
>= Supérieur ou Égal
<= Inférieur ou Égal

Opérateurs logiques

Opérateur Fonction
and Vrai si les deux conditions sont vraies
or Vrai si au moins une condition est vraie
not Inverse la valeur (vrai devient faux et vice versa)
